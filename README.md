# ShopSphere - MERN E-Commerce Platform

🚀 Live Demo: https://shopsphere-mern.vercel.app

## Overview

ShopSphere is a full-stack e-commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The platform provides a seamless shopping experience with secure authentication, product management, shopping cart functionality, and an admin dashboard.

## Features

* User Registration and Login
* JWT Authentication with Secure Cookies
* Role-Based Access Control (Admin/User)
* Product Management
* Shopping Cart
* Category-Based Product Browsing
* Responsive User Interface
* MongoDB Database Integration
* Redis Integration
* RESTful API Architecture
* Deployed Frontend and Backend

## Tech Stack

### Frontend

* React.js
* Vite
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Redis
* JWT Authentication
